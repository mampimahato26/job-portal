# 💼 Job Portal

A full-stack **MERN Stack Job Portal** that connects students/job seekers with recruiters. This platform enables recruiters to post and manage job openings while allowing students to search, apply, and track job applications through a clean and responsive interface.

---

## 🌐 Deployed Application

https://job-portal-af9e.onrender.com

---

## 📖 About the Project

The Job Portal is a web-based recruitment platform developed using the MERN Stack. It provides a seamless experience for both students and recruiters by offering secure authentication, job management, company management, and application tracking in one platform.

---

## ✨ Features

### 👨‍🎓 Student
- Register and Login
- Browse Available Jobs
- Search and Filter Jobs
- Apply for Jobs
- View Applied Jobs
- Update Profile
- Upload Profile Picture

### 🏢 Recruiter
- Register and Login
- Create and Manage Companies
- Post New Jobs
- Update Job Details
- View Applicants
- Manage Company Information

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/mampimahato26/job-portal.git
```

### Navigate to the Project

```bash
cd job-portal
```

### Install Dependencies

#### Backend

```bash
npm install
```

#### Frontend

```bash
cd frontend
npm install
```

---

## ▶️ Run the Project

### Start Backend

```bash
npm run dev
```

### Start Frontend

```bash
cd frontend
npm run dev
```
